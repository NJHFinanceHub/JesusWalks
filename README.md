# The Nazarene
Biblical action RPG in Unreal Engine 5.

## Active project
- Primary runtime: `unreal/TheNazareneAAA`
- Supported implementation language/tooling: UE5 C++, Blueprint, Data Assets, Behavior Trees, Animation Blueprints, Niagara, UMG, and Unreal config/assets.
- Legacy non-UE implementation artifacts are retired and should not be used for ongoing development.

## Unreal quick start
1. Install Unreal Engine 5.4+ and Visual Studio with the C++ toolchain.
2. Open `unreal/TheNazareneAAA/TheNazareneAAA.uproject`.
3. Generate project files if prompted, then build.
4. Launch in Unreal Editor and run PIE.
